# Customer Segmentation & Retention Analysis
Online Retail II — Commercial Customer Analytics End-to-end customer analytics on 1M+ transactions from a UK wholesale retailer (2009–2011). Covers RFM segmentation, cohort retention, CLV estimation, repeat purchase analysis, and churn proxy — with domain-aware thresholds reflecting wholesale buying behaviour.
Kaggle dataset link: [here](https://www.kaggle.com/datasets/mashlyn/online-retail-ii-uci)
