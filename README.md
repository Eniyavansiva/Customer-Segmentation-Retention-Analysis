# Customer Segmentation & Retention Analysis
Online Retail II — Commercial Customer Analytics on transactions from a UK wholesale retailer (2009–2011).   
Covers EDA, RFM segmentation, cohort retention, CLV estimation, repeat purchase analysis, and churn proxy   
with domain-aware thresholds reflecting wholesale buying behaviour.
Kaggle dataset link: [here](https://www.kaggle.com/datasets/mashlyn/online-retail-ii-uci)
